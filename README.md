# Campo Minado

Nesta aula, ministrada pela instituição <i>Senac EAD</i>, onde estou lecionando o <u>Segundo Semestre de Análise e Desenvolvimento de Sistemas</u>, o professor passou parte do código de um antigo jogo de computador chamado "Campo Minado", e o desafio era incluir um <b>contador de jogadas.</b>

Incluí uma regressiva no contador, onde a cada jogada, o usuário verá quantas jogadas ainda restam para tentar achar a Bandeirinha.
